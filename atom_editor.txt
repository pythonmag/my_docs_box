autocomplite-python
-------------------
автоподстановка синтаткиса питон
предлагает встроенные переменные, методы, функции, классы


django-wrap-i18n
----------------
Wraps selected text in {% trans "text" %}, _("text") or gettext("text") according to file extension.
If no selection - inserts import:
from django.utils.translation import ugettext_lazy as _
or
{% load i18n %}
